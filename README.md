В этом проекте отрабатывается работа с коллекциями. Изучаются способы построения и обхода деревьев.

Описание<br>
Вычислитель отличий – программа, определяющая разницу между двумя структурами данных. <br>
Это популярная задача, для решения которой существует множество онлайн-сервисов, например: http://www.jsondiff.com/. <br>
Подобный механизм используется при выводе тестов или при автоматическом отслеживании изменении в конфигурационных файлах.<br>

Возможности утилиты:<br>

Поддержка разных входных форматов: yaml и json<br>
Генерация отчета в виде plain text, stylish и json<br>
